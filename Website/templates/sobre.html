Sobre

Este website foi desenvolvido como parte complementar ao projeto académico "Previsão de Abandono para Serviços de Telecomunicações", realizado no âmbito da Unidade Curricular de Projeto I do curso de Engenharia Informática, na Escola Superior de Tecnologia do Instituto Politécnico de Castelo Branco.

O objetivo principal desta aplicação web é demonstrar a aplicação de algoritmos de Machine Learning na previsão de customer churn (o abandono de clientes) no setor das telecomunicações. Através do uso de dados reais e simulações com modelos como Random Forest, SVM, XGBoost, entre outros, o projeto identificou os modelos mais eficazes para a previsão comportamentos de abandono.

Neste website, é possível explorar de forma interativa os conceitos e resultados obtidos ao longo do projeto, bem como testar o funcionamento de um sistema de previsão de churn com base em dados de clientes simulados. A intenção é mostrar de forma acessível como a Inteligência Artificial pode ser uma aliada poderosa na retenção de clientes e na melhoria dos serviços oferecidos.

Este projeto foi realizado por Francisco Gonçalves e foi orientado pela Professora Doutora Ana Paula Neves Ferreira da Silva e pelo Professor Doutor Arlindo Ferreira da Silva.
